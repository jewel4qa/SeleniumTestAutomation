Useful command for git:

cd- change dir
pwd-> current dir
ls- list of dir in current dir

cd .. - go one step back

cd / - root dir

git clone https://github.com/jewel4qa/SeleniumTestAutomation.git --> clone any project from github

git branch -- to check current branch info

git checkout <branch name> --> change branch

git clone -b <branch name> https://github.com/jewel4qa/SeleniumTestAutomation.git
